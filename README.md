# Lezbau Calc

#### Create "virtualenv"

```bash
virtualenv lezbaucalc
```

#### Clone repository into created "lezbaucalc" directory

```bash
cd lezbaucalc
git clone https://github.com/IWUGERMANY/lezbaudata .
```

### Directory structure

```
.
├── README.md
├── bin
├── docs
├── lezbaucalc
├── lib
├── pyproject.toml
├── pyvenv.cfg
├── requirements.txt
├── share
└── tests

```

### Restore project requirements

```
pip install -r requirements.txt
```

### Run tests ... 

```
pytest
```

### Done. 👏
